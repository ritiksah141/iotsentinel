# IoTSentinel

**Educational IoT Network Security Monitor with Machine Learning Anomaly Detection**

## Project Overview

IoTSentinel is a Raspberry Pi-based network security monitoring system designed to detect anomalous behavior in home IoT devices using unsupervised machine learning. The project emphasizes educational transparency, explaining security alerts in plain English for non-technical users.

## Features

- Real-time network traffic monitoring
- Unsupervised ML anomaly detection (Isolation Forest & Autoencoder)
- User-friendly web dashboard
- Educational alert explanations
- Device discovery and tracking
- Privacy-focused (on-premise processing only)

## Technology Stack

- **Hardware**: Raspberry Pi 5 (4GB RAM)
- **Language**: Python 3.11
- **ML**: scikit-learn, TensorFlow
- **Dashboard**: Dash/Plotly
- **Database**: SQLite
- **Network**: Pcap, dpkt, Zeek
